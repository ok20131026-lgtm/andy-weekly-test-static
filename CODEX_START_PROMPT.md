# Andy Weekly Test Maker 유지보수 지시문

이 앱은 ESL Rainbows Red Vol.2 Unit 1~12 위클리 테스트용입니다.

필수 유지 조건:
1. Unit 1~12 전체를 유지한다.
2. 유닛별 20문항을 유지한다.
3. 유닛별 구성은 영영정의형 8문항, 빈칸완성형 6문항, 문장배열형 6문항이다.
4. 모든 문항은 4지선다 객관식이다.
5. 영영정의형의 hintKo는 엑셀 파일 `앤디_RED_단어장_Unit1-12_B5가로출력_완성2.xlsm`의 `영영해석` 열 값이어야 한다.
6. 영영정의형은 정답 후 해설을 표시하지 않는다.
7. 빈칸완성형과 문장배열형에는 힌트 버튼을 표시하지 않는다.
8. 홈 / 이전 문제 / 다음 문제 기능은 유지한다.
