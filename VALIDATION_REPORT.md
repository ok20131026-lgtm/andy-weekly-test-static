# 검수 결과

## 데이터 검수

- 전체 유닛: 12개
- 전체 문항: 240문항
- 유닛별 문항: 20문항
- 유닛별 구성: 영영정의형 8 + 빈칸완성형 6 + 문장배열형 6
- 전체 문항 4지선다 객관식 유지
- 영영정의형 힌트: 엑셀 `영영해석` 기준 유지
- 영영정의형 정답 후 해설 삭제 유지

## UI 수정 검수

- 결과 화면 유형별 오답 현황 표시 완료
- 유형별 정답 / 오답 / 미답 수 표시 완료
- 정답률 네모박스 높이 축소 완료
- 정답률 네모박스 색상: 어두운 남색에서 밝은 파란색 계열로 변경
- 유형별 카드 색상: 영영정의형 주황/노랑, 빈칸완성형 하늘/민트, 문장배열형 보라/분홍 계열 적용
- 홈 버튼 유지
- 오답만 다시 풀기 / 오답 목록 보기 기능 유지
