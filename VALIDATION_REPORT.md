# 검수 결과

## 문항 구성

- 전체 유닛: 12개
- 전체 문항: 240문항
- 영영정의형: 96문항
- 빈칸완성형: 72문항
- 문장배열형: 72문항
- 빈칸완성형 시각 단서 반영: 72/72문항
- 모든 빈칸완성형 visualClueEmoji 필드 생성: 완료
- 모든 문항 4지선다 확인: 완료

## 구현 반영

- 빈칸완성형 문제 화면에 `그림 단서` 영역 추가
- `imagePath`가 있으면 실제 이미지 표시
- `imagePath`가 없으면 `visualClueEmoji` 표시
- wordBank 유지
- 결과 화면 유형별 오답 현황 유지

## 주의

실제 시험지 그림과 1:1 동일한 이미지를 쓰려면 각 빈칸완성 문항별 이미지 파일을 추가한 뒤 `imagePath`를 연결해야 합니다.
