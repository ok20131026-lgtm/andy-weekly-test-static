# Andy Weekly Test Maker

ESL Rainbows Red Vol.2 앤디 위클리 테스트 인터랙티브 앱입니다.

## 최종 반영 기준

- Unit 1~12 전체 구성
- 유닛당 20문항
- 단어 문제: Unit별 단어장 Day 1-2 / Day 3-4 세트에서 영영정의 8문항
- 교재 문제: Weekly Test 범위인 Day 1 Reading, Day 3 Reading만 사용
- 빈칸완성형 6문항: Day 1·Day 3 Reading/Wrap Up 기반 그림 단서 포함
- 문장배열형 6문항: Day 1·Day 3 Reading 핵심 문장 기반
- 전 문항 4지선다 객관식
- 영영정의 힌트는 영영정의의 한글해석
- 결과 화면에서 유형별 정답/오답/미답 표시
