# Andy Weekly Test Maker

ESL Rainbows Red Vol.2 Unit 1~12 위클리 테스트 인터랙티브 앱입니다.

## 최종 반영 기준

- Unit 1~12 전체 구성
- 유닛당 20문항
- 유닛별 구성: 영영정의형 8문항 + 빈칸완성형 6문항 + 문장배열형 6문항
- 모든 문제는 4지선다 객관식
- 영영정의형은 정답 후 해설을 표시하지 않음
- 영영정의형 힌트는 `앤디_RED_단어장_Unit1-12_B5가로출력_완성2.xlsm`의 `영영해석` 열을 그대로 사용
- 빈칸완성형, 문장배열형에는 힌트 없음
- 홈 / 이전 문제 / 다음 문제 기능 유지

## 실행

```bash
npm install
npm run dev
```
