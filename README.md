# Andy Weekly Test Maker

ESL Rainbows Red Vol.2 Unit 1~12 위클리 테스트 인터랙티브 앱입니다.

## 최종 반영 기준

- Unit 1~12 전체 구성
- 유닛당 20문항
- 영영정의형 8문항 + 빈칸완성형 6문항 + 문장배열형 6문항
- 모든 문제는 4지선다 객관식
- 영영정의형 힌트는 엑셀 `영영해석` 열 기준
- 영영정의형은 정답 후 해설 삭제
- 결과 화면은 유형별 정답 / 오답 / 미답 수 표시
- 결과 화면은 초등학생용 밝은 색상 디자인 적용
- 빈칸완성형은 그림 단서가 있어야 풀 수 있으므로, 각 문항에 시각 단서 영역을 추가함

## 빈칸완성형 그림 처리

현재 수정본은 빈칸완성형 72문항 전체에 `visualClueEmoji`와 `visualClueAlt`를 넣었습니다.
실제 시험지 그림 파일이 확보되면 각 문항의 `imagePath` 필드에 `/images/...` 경로를 넣으면 실제 그림으로 바로 교체할 수 있습니다.
