# Andy Weekly Test Maker

ESL Rainbows Red Vol.2 Unit 1~12 위클리 테스트 인터랙티브 앱입니다.

## 최종 반영 기준

- Unit 1~12 전체 구성
- 유닛당 20문항
- 유닛별 구성: 영영정의형 8문항 + 빈칸완성형 6문항 + 문장배열형 6문항
- 모든 문제는 4지선다 객관식
- 영영정의형 힌트는 엑셀 단어장의 `영영해석` 기준
- 영영정의형 정답 후 해설 삭제
- 결과 화면에 유형별 정답 / 오답 / 미답 현황 표시
- 결과 화면 정답률 박스 높이 축소
- 결과 화면 색상은 초등학생용 밝은 색상으로 조정

## 실행

```bash
npm install
npm run dev
```
